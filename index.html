Analytical Engine

A Java simulation of the Analytical Engine, a mechanical computer devised by Charles Babbage in the mid-1800s but never built.

The original source code for this package was written by John Walker, and is available from this website. That software is in the public domain, according to this website.

This package is distributed under the terms of the GNU General Public License version 3.

Getting the simulator

The source code for the Analytical Engine simulator is available from GitHub. 

git clone https://github.com/jfinkels/analyticalengine

Building from source

This project uses Maven for build management and requires Java 1.7 to compile.

To create a shell script that runs the simulator, run 

mvn package appassembler:assemble

This produces a script that executes the simulator at target/appassembler/bin/analyticalengine.

For more information, visit the GitHub page.

Running an Analytical Engine program

analyticalengine myprogram.ae

Programming the Analytical Engine

The programming language for this machine is arcane, but relatively similar to modern machine languages. For more information on the instruction set and how to write programs for this machine, see this website.

Each instruction is called a "card", since the original Analytical Engine would have programmed with punched cards.

Store the number 123 in memory location 1. 

N1 123

Lines beginning with a space or a period are comments. A period after another instruction indicates a comment. 

   This program does foo and bar.

N1 123

. This is another comment

N2 456

The P instruction does not print directly to standard output; the attendant receives all printed strings and makes them all available at once after the program has terminated.
